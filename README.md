# Personal Intro Landing Page

## Project Overview

This project is a personal introduction landing page created using Lovable AI. The page presents my profile as an aspiring frontend developer and includes a hero section with my name, a professional subtitle, and a call-to-action button.

## Features

* Hero section with my name
* Professional frontend developer subtitle
* "View My Projects" button
* Responsive layout
* Customized color theme
* Modern user interface

## Manual Improvement

After exporting the project and opening it in VS Code/Cursor, I manually modified the background color to black to make it look more appealing to the eye.

## Tools Used

* Lovable AI
* VS Code
* HTML
* CSS
